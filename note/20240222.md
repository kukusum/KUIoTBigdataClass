- 2장 데이터베이스 모델 설명과 진행.
- 윈도우 DBMS에 우분투에서 접근.
    - 윈도우 DBMS에 외부 IP접근 가능한 USER 생성. 
    - 윈도우 방화벽에 3306포트 규칙생성.

- 우분투 DBMS에 윈도우에서 접근.
    - 우분투 DBMS에 외부 IP접근 가능한 USER 생성.
    - 우분투 방화벽에 전체포트 열기.(개별포트 가능하지만 나중에 추가학습예정)
    - 우분투 DBMS에 데이터 저장하고 우분투에서 데이터 확인가능.
